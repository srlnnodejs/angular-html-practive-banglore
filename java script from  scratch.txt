Quantifiers
=============
n+->matches any string that contains at least one n
isme n kiya hai a hoskata hai b hoskta hai kuch bhi hoskta hai idhar pe is me check

agge tak jaiga

n{x}->matches any string that contains a sequence fo X n's
n{x,y}->matches any string that conatins a sequence of x to y n's
n{x,}->matches any string that contaings a sequence of atleast X n's
